# ChatPlugin

---
## Building

`1)`: `./gradlew build`
`2)`: Go to `build/libs/` and copy `ChatPlugin.jar`
`3)`: Put `ChatPlugin.jar` in the `plugins` folder
`4)`: Start the server, do `/papi ecloud download Player` and `/papi reload`
`5)`: Configure the plugin to what you want.
